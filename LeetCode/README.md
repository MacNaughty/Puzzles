# LeetCode Solutions

A personal collection of LeetCode problems solved in multiple languages, primarily python. Each folder contains the problem statement, one or more solutions, and (where present) unit tests.

## Overview

- **89 problems** solved across Python, Kotlin, Go, and JavaScript
- **Folder naming:** `NNNN-problem-slug/` — zero-padded ID + kebab-case slug matching the LeetCode URL
- **Per-problem README:** each folder has a `README.md` with the problem title and a link to LeetCode

## Running the code

**Python** — most solutions include `unittest` cases. From the problem folder:

```
python -m unittest <solution_file>.py
```

**Go** — solutions with `_test.go` files:

```
go test ./<folder>
```

**Kotlin / JavaScript** — run with your standard toolchain (`kotlinc`, `node`).
